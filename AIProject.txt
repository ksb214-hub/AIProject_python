전체 흐름
    크롤링 실행 → 데이터 출력 → 사용자 입력 → 추천 실행 → 결과 출력


목표
    전체 시스템을 실행하고
    크롤링 + 추천 기능을 통합한다.

사용 라이브러리
	•	Crawling 모듈
	•	recommend 모듈


순서도
    1. 검색어 입력
    2. 크롤링 실행
        crawl(search)
    
    3. 데이터 출력
    4. 사용자 재료 입력
    5. 추천 시스템 실행
        recommend_recipe()
    
    6. 추천 결과 출력

핵심 포인트

    ✔ 컨트롤러 역할
        •	전체 흐름 제어
    ✔ 모듈 통합
	    •	Crawling + Recommend 연결
    
최종 총정리
    driver → Crawling → clean → DataFrame → recommend → scheduler

핵심 한 줄
    “이 프로젝트는 크롤링 + 데이터 처리 + 추천까지 이어지는 완전한 데이터 파이프라인이다”
