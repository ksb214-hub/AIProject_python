recommend.py

전체 흐름
    DataFrame 입력 → 레시피별 그룹화 → 유사도 계산 → 정렬 → 추천 출력

목표
    사용자 재료를 기반으로
    가장 적합한 레시피를 추천한다.

사용 라이브러리
	•	pandas

순서도
    1. DataFrame 입력
        •	레시피 / 재료 구조
    2. 레시피별 그룹화
        df.groupby("레시피")
    3. 유사도 계산
        교집합 계산
            set(user_ingredients) & set(ingredients)
        
        유사도 공식
            유사도 = (겹치는 재료 수) / (전체 재료 수)
    
    4. 결과 정렬
        sorted(..., reverse=True)

    5. 추천 결과 반환

핵심 포인트
    ✔ 유사도 기반 추천
        •	간단하지만 효과적인 방식


확장 가능
	•	TF-IDF
	•	임베딩 기반 추천
