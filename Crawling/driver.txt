driver.py

전체 흐름
    옵션 설정 → WebDriver 생성 → 반환

목표
    Selenium WebDriver 설정을 모듈화하여
    크롤링 코드에서 재사용 가능하도록 한다.


사용 라이브러리
    •	Selenium


순서도

1. 옵션 객체 생성
    Options()

2. 실행 옵션 설정
	•	headless 모드 설정 (백그라운드 실행)
	•	no-sandbox 설정
	•	GPU 비활성화

3. driver 생성
    webdriver.Chrome(options=options)

4. driver 반환
	•	다른 모듈에서 사용 가능하도록 반환

    핵심 포인트
    ✔ headless 모드
        •	브라우저 UI 없이 실행
        •	속도 향상 및 리소스 절약

    ✔ 모듈화
	    •	Crawling.py에서 재사용 가능
	    •	유지보수 용이
