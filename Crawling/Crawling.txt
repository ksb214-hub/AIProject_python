Crawling.py
전체 흐름
    검색 → 리스트 수집 → 선택 → 상세 → 재료 추출 → 정제 → DataFrame

목표
    웹스크랩핑을 통하여 데이터를 수집하는 것을 목표로 한다.

사용 언어
    python

사용 라이브러리 
    Selenium, beautifulsoup, pandas, time, re

https://www.10000recipe.com/recipe/list.html?q=

레시피 개수 selector
이 구조는 사이트 변경되면 깨질 수 있음 → 나중에 class 기반 selector로 개선 추천
#contents_area_full > ul > div > b

url selector
#contents_area_full > ul > ul > li

재료 selector
    #divConfirmedMaterialArea > ul:nth-child(1) > li

순서도 
1. 검색어 입력(검색어 변수: search)   
2. driver 셋팅(백그라운드에서 driver 실행) 
3. driver.get(첫 번째 파싱 url + search)
4. 레시피 개수 selector 사용해서 갯수 파싱하기 
5. 레시피 개수가 1개 이상일 경우 
    5-1. for in 문 
        5-1-1. selector: div > a 
            url href 파싱하기
        5-1-2. selector: div.common_sp_caption > div.common_sp_caption_tit.line2
            title text 파싱하기 
        5-2. title, href list append하기

5. 레시피 개수가 0개 미만 일 경우
    5-2. 해당 레시피가 존재 하지 않습니다.
        
6.href, title 결과 출력하기 
7. input으로 원하는 title 입력하기(0은 종료)
    7-1. 0 입력시
        driver 종료
8. driver.get(두번째 파싱 url + (원하는 title index)href)
9. beautifulsoup html파싱하기 
10. 재료 selector for in 문
    10-1. 만약 구매 단어 있으면
        구매 단어 제거 후 재료 
11. 재료 list 형태로 append
12. 재료 list pandas 형태로 변환 
13. print(pandas 형태) 출력
