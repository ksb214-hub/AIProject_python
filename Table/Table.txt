Table.py 설계 문서
1. 개요 (Overview)
    Table.py 모듈은 식자재 도매 웹사이트에서 식자재 가격 데이터를 수집하는 웹 스크래핑 모듈이다.
    사용자가 입력한 식재료를 기반으로 웹페이지에서 가격 정보를 포함한 표(Table) 데이터를 수집하고 이를 데이터 형태로 가공한다.
    수집된 데이터는 이후 **데이터 분석 및 가격 시각화(그래프)**에 활용된다.
    
    이를 통해 다음과 같은 장점을 얻을 수 있다.
    → 식자재 가격 데이터 자동 수집
    → 가격 데이터 분석 가능
    → 식자재 가격 변동 시각화


2. Table 모듈 구조
    프로젝트에서 Table 모듈은 다음 구조로 구성된다.
        plaintext
            AIProject1.py
            │
            ├─ Table/
            │   │
            │   └─ Table.py


    각 파일의 역할은 다음과 같다.
            파일	        역할
        Table.py	    식자재 가격 데이터 웹 스크래핑 및 데이터 처리

3. Table.py 역할
    Table.py 모듈은 다음과 같은 기능을 수행한다.

    → 웹 페이지에서 HTML 표(Table) 데이터 수집
    → 식자재 가격 데이터 추출
    → 불필요한 문자열 정제
    → Pandas DataFrame 생성
    → 가격 데이터 시각화 (그래프 출력)


4. 사용 기술 (Technology)
    Table 모듈에서는 다음 라이브러리를 사용한다.
        Python
            Pandas
            matplotlib
            re

    각 라이브러리의 역할은 다음과 같다.

            라이브러리	    역할
        Pandas	        데이터 구조화 및 분석 
        matplotlib      데이터 시각화 
        re	            문자열 정제


5. HTML Table 데이터 수집
    웹페이지에는 식자재 가격 정보가 HTML Table 형태로 제공된다.
        예시 구조
            html
                <table>
                    <tr>
                        <th>품목</th>
                        <th>가격</th>
                    </tr>
                    <tr>
                        <td>돼지고기</td>
                        <td>12,000원</td>
                    </tr>
                </table>
        Table.py는 이러한 HTML Table 데이터를 추출하여 분석 가능한 데이터 구조로 변환한다.


6. Table 데이터 파싱
    웹페이지에서 Table 데이터를 가져오기 위해 Pandas의 HTML 파싱 기능을 사용한다.
        예시 코드
            Python
                pd.read_html(url)
            이 함수는 웹페이지에 포함된 모든 Table을 자동으로 추출한다.

7. 데이터 추출

    Table 데이터에서 다음 정보를 추출한다.
    → 식자재 이름
    → 식자재 가격

    예시
        품목      가격
        돼지고기  12000
        닭고기    9000
        양파      2000


8. 데이터 정제
    수집된 가격 데이터에는 다음과 같은 문자열이 포함될 수 있다.

    예시
        12,000원
    이를 정제하기 위해 문자열 처리를 수행한다.

    예시 코드
        Python
            price = re.sub("[^0-9]", "", price)
    
    정제 결과   
        12000


9. DataFrame 생성
    정제된 데이터를 Pandas DataFrame 형태로 변환한다.

        예시
            재료      가격
            돼지고기   12000
            닭고기     9000
            양파       2000

        DataFrame을 사용하면 데이터 분석 및 시각화가 용이해진다.

10. 데이터 시각화
    Table.py 모듈은 수집된 가격 데이터를 기반으로 그래프를 생성한다.

    사용 라이브러리
        matplotlib
    
    예시 그래프
    → 식자재 가격 비교 그래프
    → 가격 변화 그래프


    예시 코드
        Python
            plt.bar(df["재료"], df["가격"])
            plt.show()

11. Table 모듈 특징
    Table 모듈은 다음과 같은 특징을 가진다.

    → 식자재 가격 데이터 자동 수집
    → HTML Table 데이터 자동 파싱
    → 데이터 정제 및 분석 가능
    → 가격 데이터 시각화 지원

    Table.py 모듈은 식자재 가격 데이터를 수집하고 분석하기 위한 데이터 수집 및 분석 모듈이다.

    이 모듈을 통해 다음과 같은 효과를 얻을 수 있다.
    → 식자재 가격 데이터 자동 수집
    → 데이터 기반 가격 분석 가능
